# 项目说明

## 一、项目定位

`cost_platform` 是基于若依前后端分离底座重构的企业级成本核算平台项目。

目标不是做单一场景工具，而是建设一套可以适应多行业、多公司、多合同、多费用类型，并支持发布治理、运行执行、结果追溯的成熟核算平台。

当前仓库已经收口为新的产品基线，后续开发统一以根目录设计文档为准。

## 二、当前技术底座

### 后端

- 基于若依 Spring Boot 3 体系
- 持久层采用 MyBatis + MyBatis-Plus 渐进混合模式
- JDK 17+
- 目录：`cost_platform_back`

### 前端

- 基于若依 Vue3 + Element Plus + Vite
- 目录：`cost_platform_front`

## 三、当前基线状态

当前基线保留两类内容：

- 企业级成本核算驾驶舱首页与品牌裁剪
