# v0.3.0 版本清单

## 版本定位

`v0.3.0` 定位为安全与发布风险硬化版本。当前版本不升为 `1.0.0`，原因是目标环境人工回归、权限回归、初始化验收和压测证据仍需要在真实环境补齐。
