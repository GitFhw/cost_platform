## 2026-03-30 主链回归补充

1. 新增阶梯规则时子表 sceneId 未继承当前费用所属场景
- 状态：已闭环
- 现象：页面已选中费用并进入“新增规则 -> 阶梯费率 -> 配阶梯 -> 保存”，后端返回“所属场景不能为空”。
- 根因：规则主表 `sceneId` 会在提交前补齐，但 `conditions/tiers` 子表未同步带出 `sceneId`，被 `@Validated` 在进入服务层前直接拦截。
- 修复位置：`cost_platform_front/src/views/cost/rule/index.vue`
- 修复说明：在 `normalizeSubmitData` 中统一为规则、条件、阶梯继承当前费用所属 `sceneId`。
- 验证结果：浏览器实操新增 `STORAGE_KEEP_TIER_RULE_01` 成功，规则总数从 3 增至 4，阶梯规则数从 0 增至 1。
