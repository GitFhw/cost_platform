# 线程七：公式实验室当前未落独立公式版本管理与模板库

## 风险描述
- 当前线程七已经补齐公式主数据、公式编码、中文公式、标准表达式、测试台和基础治理能力。
- 但公式仍然只有“当前态”维护，没有独立的公式版本台账、历史版本回看、回滚和模板库能力。
- 当同一公式需要长期演进、复用模板或对比历史变更时，现阶段仍要依赖发布快照和人工说明补足。

## 当前影响
- 公式实验室可用，但还不是完整的资产版本中心。
- 后续做公式模板沉淀、历史回放和跨场景复用时，治理深度不足。

## 后续计划
- 建议在线程八到线程九之间补齐公式版本台账、版本差异和模板库。
- 发布治理后续可继续复用发布快照，但公式中心仍需自己的资产版本视角。
