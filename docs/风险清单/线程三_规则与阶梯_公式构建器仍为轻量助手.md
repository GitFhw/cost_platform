# 线程三风险：规则与阶梯 - 公式构建器仍为轻量助手

## 功能归属
- 线程三：规则与阶梯
- 模块：规则中心 / 公式规则

## 当前现状
- 规则中心已经支持公式规则录入、命名空间提示、变量令牌和常用函数快捷插入。
- 当前“表达式助手”仍然是轻量辅助输入，不是完整的图形化公式构建器。

## 风险描述
- 复杂公式仍需要业务人员手工组织表达式，出错成本高于详细设计中“复用公式构建器”的理想形态。
- 对括号闭合、函数参数个数、命名空间引用正确性的校验仍主要依赖后续运行链或表达式引擎。

## 建议
- 后续在线程三持续整改或线程五联调前，补齐公式语法预校验和更强的构建器交互。
- 如变量中心已有成熟公式构建器组件，可继续抽取为规则中心共用组件。
