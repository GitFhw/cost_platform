# 风险记录

- 线程：线程二
- 功能：共享影响因素模板
- 风险等级：中

## 描述

线程二本轮已经补了共享影响因素模板能力，支持仓储保管、`zx.*`、`sg.*` 等模板在变量中心直接应用。

当前实现采用的是“内置模板库”方式：

- 模板内容写在服务端代码中
- 适合快速复用和命名空间口径收敛
- 但还不是业务可维护的正式模板管理台

## 影响

- 当前足以支撑线程二整改和线程三前的变量复用治理。
- 如果后续业务需要自行维护、审计、发布模板，现状还不够。

## 建议

- 线程二阶段保持内置模板即可，不必越界做完整模板中心。
- 若后续业务要长期维护共享模板，应在后续治理阶段再建设模板主数据、模板台账和模板发布能力。
