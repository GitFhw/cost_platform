# 风险记录

- 线程：线程二
- 功能：变量中心页面与治理能力
- 风险等级：中

## 描述

当前变量中心主链可用，但与详细设计和页面原型相比仍有缺口，尚未完全达到“线程二彻底实现”的标准。

已识别差距：

- 页面缺少原型中强调的指标卡片区域。
- 页面未形成清晰的变量详情抽屉表达。
- 导入变量、导入预览与校验报告尚未落地。
- 变量复制、共享影响因素模板等能力尚未落地。
- 接口鉴权与字段映射配置未在页面层清晰建模。

## 影响

- 业务人员能维护基础变量，但无法完整完成详细设计中要求的批量治理、接入治理和复用治理。
- 后续线程三到线程五若直接依赖当前变量页面，会放大配置理解偏差。

## 建议

- 整改线程二时先逐项对照“变量关键能力”和“变量中心原型”补齐，再继续推进后续线程。
- 页面侧要显式区分输入变量、字典变量、接口变量、第三方变量、公式变量的差异化录入结构。
