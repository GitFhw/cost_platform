# v0.2.0 对比验证清单

## 1. 目标

本清单用于在 `v0.2.0` 主线落位完成后，执行与 `v0.1.0` 的压测与结果对比，形成可追溯的放行依据。

## 2. 依赖模板

- [百万级压测方案与报告模板.md](/D:/Desktop/cost_platform/docs/百万级压测方案与报告模板.md)
- [目标环境压测执行手册.md](/D:/Desktop/cost_platform/docs/目标环境压测执行手册.md)
- [目标环境压测取证SQL.sql](/D:/Desktop/cost_platform/docs/目标环境压测取证SQL.sql)
- [正式核算上线回归清单.md](/D:/Desktop/cost_platform/docs/正式核算上线回归清单.md)

## 3. 压测对比执行

1. 选定基线口径：明确 `v0.1.0` 通过的压测场景、数据规模、并发配置与期望指标。
2. 在 `v0.1.0` 环境按基线口径跑一轮压测，记录吞吐、总耗时、失败分片数、稳定性指标。
3. 在 `v0.2.0` 环境按同口径跑一轮压测，记录同维度指标。
4. 使用取证 SQL 拉取结果台账/任务维度统计，并在报告模板中形成对比结论。

## 4. 变量/规则结果对比

1. 选定对比范围：场景、账期、费用、变量与规则口径保持一致。
2. 生成 `v0.1.0` 与 `v0.2.0` 的输出台账快照（按任务或账期维度）。
3. 对比关键字段：费用金额、规则命中、变量取值、追溯链路摘要。
4. 记录差异原因，区分：旧实现缺陷被修正导致的合理差异；新实现引入的非预期差异。
5. 形成最终差异结论与放行建议。

## 5. 产出物

- 压测对比报告
- 变量/规则结果对比记录
- 差异归因说明
- 放行结论
