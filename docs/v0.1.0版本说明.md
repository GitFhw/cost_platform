# v0.1.0 版本说明

## 1. 版本定位

`v0.1.0` 是企业级核算平台第一阶段可运行版本，定位为：

- 核算主链已经贯通
- 场景、费用、变量、规则、发布、运行、结果台账已经形成平台雏形
- 第三方变量接入能力已经具备平台化基础
- 初始化脚本、治理能力、压测与调度能力已经具备第一阶段交付口径

本版本不是最终架构完成版，而是企业级核算平台的第一阶段正式基线版本。

## 4. 当前版本未完成但已明确的方向

这些事项不再视为 `v0.1.0` 阻塞项，但属于明确的后续演进方向：

- 第三方接入能力继续平台化
- 变量解析链标准化
- 公式依赖图与环检测平台化
- 费用节点执行器抽象
- 发布校验链抽象
- 多节点调度深化
- 目标环境压测固化

## 5. 版本边界

`v0.1.0` 的边界如下：

- 已经可以作为企业级核算平台第一阶段基线版本继续开发
- 不承诺当前架构已经完成最终拆层
- 不承诺第三方网络环境问题已在平台内部彻底解决
- 不承诺多节点调度已经完成生产级全闭环

## 6. 版本要求

从 `v0.1.0` 开始：

- 主线开发以 `main` 为稳定基线
- 后续开发需求以 `0.2.0` 计划文件为执行约束
- 不再接受绕开设计基线的临时型结构改造
