# cost_platform

企业级核算平台重构底座。

## 目录结构

- `cost_platform_back`：基于若依后端底座的核算平台服务端工程
- `cost_platform_front`：基于若依 Vue3 的核算平台前端工程
- 根目录文档：产品设计、SQL、ER 图、页面原型、缓存规范和开发验证规范

## 当前目标

基于若依前后端分离底座，逐步重构一套企业级核算平台，先把业务人员最常维护的主链做清楚：

- 场景中心
- 费用中心
- 规则中心
- 阶梯规则
- 发布、追溯与治理能力

## 当前重构主线

1. 独立系统字典维护
2. 配置中心业务域字典化
3. 费用中心对齐老项目“基础费目维护”
4. 规则中心逐步改造成“费用 -> 规则 -> 阶梯”的维护模式
