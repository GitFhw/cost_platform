# cost_platform

企业级核算平台重构底座。

## 目录结构

- `cost_platform_back`：若依后端底座，后续承载核算平台后端能力
- `cost_platform_front`：若依 Vue3 前端底座，后续承载核算平台前端页面
- `docs`：项目文档、概念说明、重构计划

## 当前目标

基于若依前后端分离底座，重新构建企业级核算平台，逐步沉淀以下能力：

- 配置中心
- 系统字典
- 变量中心
- 费用主数据
- 费率规则与阶梯规则
- 发布、追溯与治理能力

## 近期重构方向

1. 独立系统字典维护
2. 配置中心业务域字典化
3. 费用主数据维护
4. 规则维护改造成“费用 -> 规则 -> 阶梯”

## 说明

当前仓库是新的产品重构工作区。后续开发、提交与文档整理，都以本仓库为主。
